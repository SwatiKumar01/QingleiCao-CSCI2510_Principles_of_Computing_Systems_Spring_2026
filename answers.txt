1. What is the return type and expected arguments for the thread entry function?

Return type: void*
Expected argument: void*

2. Program output for creating one thread:

Hello from thread!
Thread finished.

3. Struct definition:

struct thread_args
{
    int arg1;
    char arg2[100];
    int ret;
};

4. Program output when passing struct values:

thread arg1: 25
thread arg2: Hello thread
Returned value: 999

5. Program output for five threads:

Thread 0 says: This is thread 0
Main got ret from thread 0 = 0
Thread 3 says: This is thread 3
Thread 1 says: This is thread 1
Thread 4 says: This is thread 4
Thread 2 says: This is thread 2
Main got ret from thread 1 = 10
Main got ret from thread 2 = 20
Main got ret from thread 3 = 30
Main got ret from thread 4 = 40

6. What do you notice about the output ordering? Why?

The output ordering changes between runs. Threads execute concurrently, and the operating system scheduler decides which thread runs first, so the order is not guaranteed.
